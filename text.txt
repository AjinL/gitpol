git info  - информация о проекте
git clone (сылка) - копировать проект из репозитория
git add (файл) - переводим в статус перед добавлением проекта
git commit -m "Комит" - создание комита
git push - закидываем в репозиторий
При первом использовании этой команды просит ввести логин и пароль от gihab
